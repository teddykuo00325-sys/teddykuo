【最簡單的啟動方式】

  1. 解壓縮 .zip 檔案到任一資料夾
  2. 雙擊  啟動凌策.bat
  3. 等待 10 秒左右，瀏覽器會自動打開
  4. 若要關閉系統，關掉出現的黑色視窗即可

═══════════════════════════════════════════════════

【事前需要安裝的東西】

  必裝 ─ Python 3.11 或以上（才能跑後端）
         下載：https://www.python.org/downloads/
         安裝時務必勾選「Add Python to PATH」

  選裝 ─ Ollama（本地 AI 模型）
         下載：https://ollama.com
         安裝後執行：ollama pull qwen2.5:7b

  - 沒裝 Ollama 也能跑，僅部分 AI 回答功能停用
  - 知識庫、CRM、報價 PDF、合規中心、官網、客戶視角、12 坪情境都能正常運作

═══════════════════════════════════════════════════

【首次啟動流程】

  第一次雙擊啟動凌策.bat 時，會自動：
    [1/4] 檢查 Python 是否存在
    [2/4] 自動安裝 Flask / reportlab / ChromaDB 等套件（約 1~3 分鐘）
    [3/4] 檢查 Ollama 狀態
    [4/4] 啟動 Flask 伺服器並開啟瀏覽器

  第二次以後啟動，會跳過套件安裝，約 10 秒內完成啟動。

═══════════════════════════════════════════════════

【瀏覽器打開後看到什麼】

  預設會進入「凌策對外官網」(http://127.0.0.1:5000/index.html)
  這是給外部潛客看的入口。

  若要看內部管理介面，從側邊欄點「🔐 內部控制台」
  或直接打開 http://127.0.0.1:5000/dashboard.html

═══════════════════════════════════════════════════

【遇到問題】

  Q1: 瀏覽器打開顯示「無法連線」
  →  黑色視窗應該有錯誤訊息，通常是 Python 或套件沒裝好。
      可以手動執行：pip install -r requirements.txt

  Q2: 所有頁面都顯示「後端未連線」
  →  Flask 沒啟動。雙擊 啟動凌策.bat 重新啟動。

  Q3: AI 指揮官、客戶模擬器的 AI 回答很慢或失敗
  →  Ollama 沒跑。到 https://ollama.com 安裝後執行：
      ollama serve
      ollama pull qwen2.5:7b

  Q4: 埠號 5000 被佔用
  →  啟動凌策.bat 會自動關掉舊的；若仍失敗，重開機。

【技術堆疊】

  後端：Flask + Ollama (Qwen 2.5 7B) + ChromaDB + SQLite
  前端：HTML + Tailwind CSS（CDN）
  PDF：reportlab + Microsoft 正黑體（Windows 內建）
  稽核：JSONL append-only 永久留存
