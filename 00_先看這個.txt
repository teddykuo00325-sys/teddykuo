【實際執行系統 — Windows 10/11 雙擊即跑】

  雙擊：啟動凌策.bat

  ⚠️ 重要：等瀏覽器自動開啟後，使用伺服器網址：
       http://localhost:5000/index.html
       http://localhost:5000/dashboard.html

     請 ★ 不要 ★ 從檔案總管雙擊 dashboard.html 開啟
     （那會變成 file:// 協定，瀏覽器 CORS 會擋 API 呼叫，
       導致「知識庫連線失敗」「行銷文案無法生成」等問題）

  啟動流程自動化：
    [1/5] 檢查 Python → 若無，透過 winget 自動安裝（需聯網）
    [2/5] 安裝必要套件（約 1 分鐘，精簡版 30MB）
    [3/5] 檢查 Ollama（選用，沒裝也能跑 80% 功能）
    [4/5] 清理舊伺服器
    [5/5] 啟動後端 → 自動開啟瀏覽器

  首次啟動約需 20~60 秒；後續啟動約 10 秒。

【常見問題】

  Q: 雙擊 bat 後黑窗立即消失
  A: 未安裝 Python。開啟「命令提示字元」，手動執行：
       python --version
     若無任何輸出，請到 https://www.python.org/downloads/ 安裝。
     安裝時務必勾選「Add Python to PATH」

  Q: 瀏覽器顯示「無法連上 127.0.0.1」
  A: Flask 還在啟動中（首次 ChromaDB 初始化慢）。等 60 秒後重新整理。
     或直接看 "LingCe Backend Server" 黑色視窗中的錯誤訊息。

  Q: 路徑含中文會不會有問題？
  A: 測試過 Desktop / 公司AI比賽 / lingce-company 路徑可以正常運行。
     若仍失敗，請將整個資料夾移到 C:\lingce-company\ 再試。

  Q: AI 指揮官、客戶模擬器的 AI 回覆失敗
  A: 未安裝 Ollama。請到 https://ollama.com 下載並執行：
       ollama pull qwen2.5:7b
